## example/src

This directory contains individual example files that show how specific widgets or features of the RadarTUI framework work.

### Available Examples

- **`align_example.dart`**: Demonstrates `Align` widget positioning.
- **`async_example.dart`**: Shows async data loading with `FutureBuilder`.
- **`button_example.dart`**: Example of how to use the `Button` widget.
- **`card_example.dart`**: Demonstrates `Card` widget layout.
- **`checkbox_example.dart`**: Example of `Checkbox` toggling state.
- **`circular_progress_example.dart`**: Shows `CircularProgressIndicator`.
- **`dashboard_example.dart`**: Complex dashboard layout combining widgets.
- **`data_table_example.dart`**: Demonstrates `DataTable` with sorting.
- **`default_text_style_example.dart`**: Shows `DefaultTextStyle` inheritance.
- **`dialog_example.dart`**: Example of modal `Dialog` with `showDialog`.
- **`divider_example.dart`**: Demonstrates `Divider` widget.
- **`dropdown_button_example.dart`**: Example of `DropdownButton` selection.
- **`expansion_tile_example.dart`**: Shows `ExpansionTile` expand/collapse.
- **`flex_example.dart`**: Demonstrates `Row` and `Column` flex layouts.
- **`focus_example.dart`**: Example of `Focus` widget and focus management.
- **`form_example.dart`**: Shows form input with validation.
- **`grid_view_example.dart`**: Demonstrates `GridView` grid layout.
- **`icon_example.dart`**: Example of `Icon` widget rendering.
- **`linear_progress_example.dart`**: Shows `LinearProgressIndicator`.
- **`menu_bar_example.dart`**: Demonstrates `MenuBar` navigation.
- **`radio_example.dart`**: Example of `Radio` button group selection.
- **`rich_text_example.dart`**: Shows `RichText` with mixed styles.
- **`shortcuts_example.dart`**: Demonstrates `Shortcuts` and `Actions`.
- **`slider_example.dart`**: Example of `Slider` value adjustment.
- **`spacer_example.dart`**: Demonstrates `Spacer` in flex layouts.
- **`sparkline_example.dart`**: Shows `Sparkline` data visualization.
- **`stack_example.dart`**: Example of `Stack` layered positioning.
- **`status_bar_example.dart`**: Demonstrates `StatusBar` widget.
- **`style_example.dart`**: Styling example using `TextStyle` and `Container`.
- **`tab_bar_example.dart`**: Example of `TabBar` navigation.
- **`textfield_example.dart`**: Example of how to use the `TextField` widget.
- **`theme_example.dart`**: Demonstrates theming with `Theme`.
- **`toast_example.dart`**: Example of `Toast` notifications.
- **`toggle_example.dart`**: Shows `Toggle` switch widget.
- **`tree_view_example.dart`**: Demonstrates `TreeView` hierarchical display.
